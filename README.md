# Transplant Stickers

An iMessage sticker pack promoting organ donation awareness and supporting the transplant community.

## Overview

This sticker pack includes 37 unique stickers featuring:
- **Anatomical organs**: Heart, kidneys, liver, lungs, and pancreas
- **Donation themes**: Living donation, paired kidney exchange, and organ donation awareness
- **Educational messages**: "Register to be an Organ Donor", "Organ Donation Saves Lives", "Save a Life"
- **Symbolic stickers**: Hands holding organs, transplanted organs with ribbons, and medical imagery

## Purpose

These stickers are designed to:
- Raise awareness about organ donation
- Support transplant recipients and living donors
- Encourage conversations about organ donation registration
- Provide a fun and meaningful way to share transplant-related messages

To prepare the sticker pack for App Store submission:

1. Update version numbers in the project settings

3. Archive the app (Product > Archive)
4. Submit via App Store Connect
